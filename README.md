# Client Commande Front

Application Angular permettant la gestion des **clients** et des **commandes**
avec authentification JWT et gestion des rôles **ADMIN / USER**.

## Fonctionnalités
- Authentification (JWT)
- Rôles ADMIN / USER
- CRUD Clients
- CRUD Commandes
- Guards Angular
- UI propre avec validation & toasts

## Lancer le projet
```bash
npm install
ng serve
